Ce este CSS ?

Lecia trecut am studiat cum s crem o pagin web cu ajutorul HTML. ns HTML este limbajul care definete structura documentului i nu ne ofer posibilitatea de a face pagina noastr s arate frumos.

n ajutor ne poate veni CSSCascading style sheets.

CSS este un standard care definete modul n care vor fi afiate elementele din pagina noastr.

Pentru nceput s crem un document HTML index.html pe care l vom folosi n calitate de baz pentru site-ul tu.

Documentul tu HTML va arta n felul urmtor:

<!DOCTYPE html>
<html>
<head>
    <title></title>
</head>
<body>
</body>
</html>

Sintaxa CSS

Selectori

Pentru a ncepe lucrul, trebuie s nelegem sintaxa CSS.

Aceasta este una simpl i arat n felul urmtor:

Aceasta const dintr-un selector i proprietile aplicate acestuia, sub form de perechi chei valoare. n calitate de selector pot servi diferite elemente HTML, tag-uri, clase i orice combinaii ntre acestea.

Sun complicat?

S vedem un exemplu. Vom crea un paragraf nou i vom seta culoarea textului s fie roie i de dimensiunea 25px. Pentru aceasta utilizm deja cunoscutul HTML: adugm tagul p n tag-ul body.

<p>Salut</p>

Acum urmeaz s stilizm HTML-ul cu CSS. Pentru aceasta vom aduga n head un tag special style, unde vom descrie stilul dorit:

<style type="text/css">
    p {
        color: red;
        font-size: 25px
    }
</style>

Acum deschide fiierul index.html n browser-ul tu preferat i vei vedea urmtorul rezultat.

Arat mai bine dect fr stil, aa-i?

Selectori utiliznd clase

Pe motiv c n calitate de selector am luat tag-ul p, deocamdat oricare tag p din documentul HTML va fi stilizat utiliznd definiia noastr n CSS. Dac dorim s aplicm un anumit stil doar elementelor care au careva proprieti comune, n calitate de selector putem s lum o clas.

De exemplu, s crem un container div de clasa my-class:

<div class='my-class'>Hello</div>

Apoi vom aduga un selector pentru clasa my-class nuntrul tag-ului style din HTML:

.my-class {
    background-color: orange;
    font-size: 30px;
}

Selectori utiliznd ID

n calitate de selector poate fi utilizat i un ID (identificator unic pentru orice element HTML).

Pentru a exemplifica, vom modifica puin div-ul din exemplul precedent: n loc de atributul class vom scrie un id al elementului:

<div id='my-id'>Hola!</div>

Iar pentru a stiliza, vom mai aduga un selector pentru id:

#my-id {
    background-color: blue;
    font-size: 40px;
}

n exemplul dat, ID-ul elementului div este my-id. Reactualizeaz pagina din browser i vei vedea urmtorul rezultat:

Selectori multipli

Pn acum definiiile noastre pentru stil erau identice pentru toate elementele cu acelai tag i clas. De aceea, pentru a putea fi i mai specifici, putem s combinm mai muli selectori ntr-unul.

De exemplu, urmtorul selector va stiliza n mod corespunztor toate elementele cu clasa my-paragraph-class ale cror printe este un element div cu clasa my-class:

div.my-class .my-paragraph-class {
    color: red;
    background-color: cian;
    text-align: center;
}
<div class="my-class">
  <p class="my-paragraph-class">Bonjour</p>
<div>

Ai remarcat structura? Dac doreti s alegi/setezi un selector pentru un element child (copil) care este nuntrul altui element parent (printe), trebuie doar s nlnui selectorul printelui i al copilului, separndu-le prin spaii:

parent child {
    key: value;
}

Cum s folosesc CSS ?

Sunt 3 metode cum poi stiliza HTML-ul cu CSS:

  1. Incluzi CSS-ul ntr-un fiier extern cu extensia .css (external style sheet).

    De asemenea, vei lega documentul CSS style.css de cel HTML, folosind tag-ul <link rel="stylesheet" type="text/css" href="style.css">. Atrage atenia la atributul href---acesta arat calea spre fiierul .css.

  2. Foloseti tagul style n head-ul al documentului HTML (internal style sheet):

    <style> p { text-indent: 25px; } </style>

n interiorul style poi lista mai multe declaraii de stil.

  1. Foloseti atributul style pentru fiecare tag din document (inline style):

<p style="text-indent: 25px">Hola!</p>

Cnd lucrezi cu stiluri trebuie s ii minte dou reguli: 1. pentru acelai selector se aplic ultimul stil declarat; 2. fiecare metod de declarare a stilului are prioritatea sa: inline style este cel mai prioritar, apoi urmeaz internal style i doar la sfrit external style.

Culori i background

Pentru a seta culoarea unui text, poi folosi proprietatea color care poate s primeasc urmtoarele tipuri de valori:

  1. numele culorii: red, green, blue;

  2. codul hexadecimal al culorii: #ff00ff;

  3. codul RBG rgb(255, 0, 0).

De exemplu, poi seta culoarea roie a textului n trei metode:

p {
    color: #ff0000;
    color: rgb(255, 0, 0);
    color: red;
}

Pentru a seta culoarea de fundal a unui element folosete proprietatea background-color:

div {
    background-color: lightblue;
}

Imagine pe fundal?

Dac doreti s plasezi o imagine pe fundal, ai la ndemn proprietatea background-image unde trebuie sa specifici adresa imaginii.

div {
    background-image: url("paper.gif");
}

Stilizarea textelor i fonturi

Fontul

Pentru a seta un font pentru text, poi folosi proprietatea font-family.

p {
    font-family: "Times New Roman";
}

Aceasta va seta fontul Times New Roman.

Dar ce dac pe calculator nu este instalat Times New Roman? Pentru a rezolva aceast problem, poi nlnui mai multe fonturi:

p {
    font-family: "Times New Roman", Times, serif;
}

Acum se va seta Times New Roman pentru elementul tu. n caz c acest font lipsete pe calculatorul utilizatorului, se va aplica fontul Times, iar dac nici acesta nu este prezent, se va recurge la familia de fonturi serif.

Mrimea fontului

Pentru a controla mrimea fontului de folos i va fi proprietatea font-size, unde poi indica n pixeli (px) nlimea textului:

h1 {
    font-size: 40px;
}

Mai multe despre fonturi poi s afli aici.

Box model

Box model este un concept care ne permite uor s vizualizm i s nelegem conceptul de margin, padding, border i width / height a unui element HTML.

n box model un element HTML este reprezentat prin nite cutii, ele fiind:

  • margin - distana n jurul elementului, este transparent;

  • border - chenarul elementului;

  • padding - distana ntre coninutul elementului i chenar;

  • coninutul - spaiul unde este afiat textul interior / imaginea.

nlimea i lungimea elementului HTML se calculeaz n dependen de cele 4 componente.

De exemplu, avem un element cu urmtoarele proprieti:

div {
    width: 320px;
    padding: 10px;
    border: 5px solid gray;
    margin: 0; 
}

Limea acestuia va fi de 350px: 320px + 20px (left + right padding) + 10px (left + right border) + 0px (left + right margin).

Despre parametrii proprietilor din box model poi citi suplimentar aici i aici.

Poziionare

Pn acum am discutat despre cum vor arta elementele paginii, dar nu am menionat n ce ordine ele vor aprea pe ecran i unde trebuie s se afle un anumit element.

Pentru aceste scopuri n CSS exist proprietatea position. Aceasta specific metoda prin care un element HTML va fi poziionat n pagin.

Exist 4 posibile valori pentru proprietatea position:

  • static---elementul este poziionat dup setarea implicit a browser-ului;

    div.static {
     position: static;
     border: 3px solid #73AD21;
    }

  • relative---elementul este poziionat relativ fa de poziia lui iniial, nsa ocupa spaiul rezervat;

div.relative {
    position: relative;
    left: 30px;
    border: 3px solid red;
}

  • absolute---elementul este poziionat absolut fa de primul printe cu poziia relative sau, dac acesta nu este, fa de elementul body al documentului HTML;

div.relative {
    position: relative;
    width: 400px;
    height: 200px;
    border: 3px solid #73AD21;
} 
div.absolute {
    position: absolute;
    top: 80px;
    right: 0;
    width: 200px;
    height: 100px;
    border: 3px solid #73AD21;
}

  • fixed---specific poziionarea fix pe pagin, asta nseamn c elementul va rmne constant pe ecran, chiar dac facem scroll.

div.fixed {
    position: fixed;
    bottom: 0;
    right: 0;
    width: 300px;
    border: 3px solid #73AD21;
}

Elemente plutitoare

Cu toate c avem deja posibilitatea s micm elementele n pagin, este foarte anevoios s aliniem elementele pe orizontal.

Soluia uoar pentru aceast problem este proprietatea CSS float. Aceasta ne permite s crem elemente plutitoare poziionate unul lng altul, orizontal. Cu ajutorul acestei proprieti putem crea galerii de imagini, liste de nouti i alte elemente.

Astfel, elementele HTML cu proprietatea float: left se vor aranja orizontal unul dup altul.

.float {
    width: 100px;
    height: 100px;
    border: 1px solid red;
    margin: 10px;
    float: left;
 }
<div class='float'></div>
<div class='float'></div>

Odat cu libertatea de a plasa elementele pe orizontal apare problema cu elementele statice care apar mai jos n documentul HTML. n mod special, dac acestea sunt statice, ele se vor suprapune cu elementele plutitoare.

De exemplu, dac la exemplul de mai sus adugm un element dup elementele cu clasa float, el se va suprapune cu elementele plutitoare.

Pentru a evita aceast suprapunere trebuie s atribuim elementului care urmeaz dup elementele plutitoare proprietatea clear: both:

.float {
    width: 100px;
    height: 100px;
    border: 1px solid red;
    margin: 10px;
    float: left;
 }
 .element {
    height: 200px;
    width: 200px;
    background-color: orange;
    clear: both;
 }
<div class='float'></div>
<div class='float'></div>
<div class='element'></div>
